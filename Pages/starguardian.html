Features

- Add Role
This feature allows users to add roles to themselves. To use this feature, simply type "add role" followed by the name of the role in the chat and the bot will add the role to you.

- Ban
This feature allows users to ban other users from the server. To use this feature, simply type "ban" followed by the username of the user you want to ban in the chat and the bot will ban the user.

- Scam Detection
This feature allows users to detect scams. To use this feature, simply type "scam" followed by the link you want to check in the chat and the bot will respond with whether or not the link is a scam.

- Kick
This feature allows users to kick other users from the server. To use this feature, simply type "kick" followed by the username of the user you want to kick in the chat and the bot will kick the user.

- RemoveRole
This feature allows users to remove roles from themselves. To use this feature, simply type "remove role" followed by the name of the role you want to remove in the chat and the bot will remove the role from you.

- Unban
This feature allows users to unban other users from the server. To use this feature, simply type "unban" followed by the username of the user you want to unban in the chat and the bot will unban the user.

- Animal Facts
This feature allows users to learn interesting facts about animals. To use this feature, simply type "animal facts" in the chat and the bot will respond with a random fact about an animal.

- Memes
This feature allows users to share memes in the chat. To use this feature, simply type "meme" in the chat and the bot will respond with a random meme.

- Horoscopes
This feature allows users to get their horoscope for the day. To use this feature, simply type "horoscope" followed by your sign in the chat and the bot will respond with your horoscope.

- Ily (No is always the response)
This feature is a joke feature that always responds with "no" when users tell the bot they love it. To use this feature, simply type "ily" in the chat and the bot will respond with "no."

- PlayCheck
This feature checks the status of Spotify, MusicBee, and Foobar2000. To use this feature, simply type "playcheck" and the bot will respond with the status of the music player.

- Bot Info
This feature allows users to get their horoscope for the day. To use this feature, simply type "horoscope" and select what your sign is and the bot will respond with your horoscope.

- Bot Invite (Invites Bot)
This feature allows users to invite the bot to their server. To use this feature, simply type "bot invite" in the chat and the bot will respond with a link that you can use to invite the bot to your server.

- Bot Ping (Server Ping)
This feature allows users to check the bot's latency. To use this feature, simply type "bot ping" in the chat and the bot will respond with its latency.

- Server Info
This feature allows users to get information about the server they are in. To use this feature, simply type "server info" in the chat and the bot will respond with information about the server, such as its name, owner, and members.

- User Info
This feature allows users to get information about another user. To use this feature, simply type "user info" followed by the username of the user you want to get information about in the chat and the bot will respond with information about the user, such as their name, role, and join date.
